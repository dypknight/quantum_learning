平衡函数：在所有2^n个输入中，恰好一半输出0，一半输出1

常函数：无论输入是什么，输出永远相同。

贡献=系数*点积结果

系数：如cx门，哪一位是控制位，控制位为0，系数为+1，控制位为1，系数为-1

点积结果：（-1）^a.z ，可能跟每一位输入x有关系，要看z哪一位为0，哪一位为1，对应为相乘求和

贡献：8种输入x对具体某一个z的贡献，可以理解为z是经典比特，x是8种量子叠加态，8种情况累加起来，再归一化 就是该z状态的振幅，即塌缩到这种z状态的概率

2026/04/01  Grover.py 搜索算法

==============================================================================
首先，先化简一下搜索模型，将所有数据存在数据库中，假设有n个量子比特，用来记录数据库中的每一个数据的索引，一共可以表示

2^n个数据，记为N个；希望搜索得到的数据有M个，为了表示一个数据是否是搜索的结果，建立函数，f(x)=0 (x不等于x0)；f(x)=1 (x=x0)

x0为搜索目标的索引值。

核心思想：不断放大正确答案的概率，压低错误答案的概率

Oracle只改相位，不直接改概率，概率=振幅的平方，Oracle只改了振幅的相位（正负）
